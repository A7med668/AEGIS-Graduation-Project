# Dataset Sources Guide

## Benign apps
استخدم F-Droid كبداية ممتازة للتطبيقات العادية والمفتوحة المصدر.

## Permission-heavy benign
اختار تطبيقات عادية لكن بصلاحيات كتير:
- app store
- file manager
- chat app
- camera/voice app
- maps app
- browser

## Suspicious / malware-like
للديمو استخدم samples تعليمية أو dummy apps. لو هتستخدم malware datasets حقيقية، خليها emulator-only.

## فولدرات المشروع
